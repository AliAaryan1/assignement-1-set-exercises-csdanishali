Essential HTML & CSS Reference

HTML Elements

Document Structure

<!DOCTYPE html>
<html lang="en">
<head>  
    <meta charset="UTF-8">
    <title>Page Title</title>
</head> 
<body>
    <!-- Content -->
</body>
</html>

Basic HTML5 document structure with essential meta tags.

Text Elements

<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>Paragraph text</p>
<strong>Bold text</strong>
<em>Italic text</em>

Common text formatting elements for content structure.

Lists

<ul>
    <li>Unordered item</li>
</ul>

<ol>
    <li>Ordered item</li>
</ol>

Create bulleted or numbered lists.

Links & Images

<a href="url">Link Text</a>
<img src="image.jpg" alt="Description">

Hyperlinks and image embedding.

Semantic Elements

<header></header>
<nav></nav>
<main></main>
<section></section>
<article></article>
<footer></footer>

HTML5 semantic elements for better structure.

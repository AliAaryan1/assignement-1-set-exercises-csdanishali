HTML & CSS Quick Reference Guide

HTML Tags

<h1> to <h6>

Defines HTML headings.

<h1>This is a heading</h1>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

<a>

Defines a hyperlink.

<a href="https://example.com">This is a link</a>

<img>

Defines an image.

<img src="image.jpg" alt="Description">

<ul> & <li>

Defines an unordered list and list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol> & <li>

Defines an ordered list and list items.

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<div>

Defines a division or section.

<div>This is a div</div>

<span>

Defines a section in a document.

<span>This is a span</span>

<table>, <tr>, <td>

Defines a table, table row, and table cell.

<table>
    <tr>
        <td>Cell 1</td>
        <td>Cell 2</td>
    </tr>
</table>

<form>

Defines an HTML form for user input.

<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</form>

CSS Properties

color

Sets the color of text.

p {
    color: blue;
}

background-color

Sets the background color of an element.

div {
    background-color: lightgrey;
}

font-size

Sets the size of the font.

h1 {
    font-size: 2em;
}

margin

Sets the margin area on all four sides of an element.

p {
    margin: 20px;
}

padding

Sets the padding area on all four sides of an element.

div {
    padding: 10px;
}

HTML & CSS Quick Reference Guide

A handy guide to commonly used HTML tags and CSS properties.

HTML Tags

1. <h1> to <h6> - Headings

Used to define headings of different levels.

<h1>Main Heading</h1>

2. <p> - Paragraph

Defines a block of text as a paragraph.

<p>This is a paragraph.</p>

3. <a> - Anchor

Creates hyperlinks to navigate to other pages or resources.

<a href="https://example.com">Visit Example</a>

4. <img> - Image

Embeds an image in the document.

<img src="image.jpg" alt="Description">

5. <ul> and <li> - Unordered List

Creates a bulleted list.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

6. <ol> and <li> - Ordered List

Creates a numbered list.

<ol>
    <li>First Item</li>
    <li>Second Item</li>
</ol>

7. <table> - Table

Defines a table structure.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

8. <div> - Division

Defines a container for grouping content.

<div>This is a container.</div>

9. <span> - Inline Container

Defines an inline container for text or elements.

<span>Highlighted text</span>

10. <form> - Form

Defines a form for user input.

<form>
    <input type="text" name="name" placeholder="Enter your name">
    <button type="submit">Submit</button>
</form>

CSS Properties

1. color

Sets the text color.

p {
    color: blue;
}

2. background-color

Sets the background color of an element.

div {
    background-color: lightgray;
}

3. font-size

Sets the size of the font.

h1 {
    font-size: 24px;
}

4. margin

Sets the outer spacing of an element.

div {
    margin: 20px;
}

5. padding

Sets the inner spacing of an element.

div {
    padding: 10px;
}

6. border

Sets the border of an element.

div {
    border: 1px solid black;
}

7. text-align

Aligns text horizontally.

h1 {
    text-align: center;
}

8. display

Specifies the display behavior of an element.

div {
    display: flex;
}
